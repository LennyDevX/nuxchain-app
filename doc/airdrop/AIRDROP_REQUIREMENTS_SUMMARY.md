# Nuxchain Airdrop - Registration Requirements Summary

## 📋 Quick Requirements Checklist

Para registrarse exitosamente en el airdrop de Nuxchain y recibir **6,000 NUX tokens**, necesitas cumplir con:

### ✅ Personal Information
- **Name**: Mínimo 3 caracteres
- **Email**: Dirección de correo válida y única (se usa para enviar tokens)

### ✅ Wallet Requirements
- **Dirección válida**: Solana address en formato Base58 (32-44 caracteres)
- **Balance mínimo**: ≥ 0.001 SOL en la wallet (protección contra bots)
- **No duplicada**: La wallet no debe estar registrada previamente

### ✅ Wallet Legitimacy (Security Analysis)
El sistema analiza automáticamente la wallet para asegurar que es legítima:

**✓ Aprobación Inmediata:**
- Wallet proveniente de exchange (CEX): Coinbase, Kraken, Binance, etc.
- Indicadores de legitimidad claros

**✓ Aprobación Probable:**
- Legacy/Old wallets: Wallets antiguas (> 30 días)
- Actividad de transacciones consistente
- Multiple token accounts
- Historial económico sano

**✗ Rechazo Probable:**
- Wallets nuevas (< 7 días) sin originación clara
- Patrones sospechosos (velocidad alta de transacciones, movimientos anormales)
- Riesgo detectado por análisis heurístico
- Balance muy bajo o insuficiente

### ⏱️ Additional Requirements
- **Tiempo de registro**: Mínimo 3 segundos desde la carga de la página (protección contra bots)
- **Email único**: No puedes usar el mismo email dos veces
- **Pool capacity**: Máximo 10,000 usuarios (FCFS - First Come, First Served)
