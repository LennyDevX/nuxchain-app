💻 COMANDOS PRINCIPALES
═══════════════════════════════════════════════════════════════════

VER ESTADO
  node AirdropsWallet.js stats
  → Muestra: usuarios, wallets válidas, capacidad pool

EXPORTAR DATOS
  node AirdropsWallet.js export
  → Genera: CSV + JSON en airdrop-exports/

VALIDAR WALLETS
  node AirdropsWallet.js validate
  → Detecta: inválidas, duplicadas, problemas

PREPARAR DISTRIBUCIÓN
  node AirdropsWallet.js distribute
  → Genera: reporte JSON completo

FORMATOS BLOCKCHAIN
  node distribute.js solana         # Solana SPL
  node distribute.js polygon        # EVM (Polygon)
  node distribute.js csv [N]        # CSV, primeros N usuarios
